# OdinGame


An app written in Odin with Raylib based on https://github.com/adam-lafontaine/SDL3Game


* [x] Keyboard input
* [x] Mouse input
* [x] Gamepad input
* [x] Load assets from a binary file in a worker thread
* [x] Process raw image data for UI
* [ ] Play music
* [ ] Play sound effects
* [ ] Generate code from raw asset files
